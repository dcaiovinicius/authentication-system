📘 API Quick actions

All endpoints comply with your Go spec:

  • POST /api/v1/register → returns tokens
  • POST /api/v1/login → returns tokens
  • POST /api/v1/refresh → revokes old, issues new
  • GET /api/v1/users → requires Bearer token
  • POST /api/v1/logout → revokes refresh token
